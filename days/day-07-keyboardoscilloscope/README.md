# KeyboardOscilloscope

Use your computer keyboard to play simultaneous synthesized sine-waves and trace their combined multi-frequency waveform on a realtime oscilloscope window.

## Hardware Concept Analog
Switches -> Signal Generator -> Oscilloscope. Analogous to a hardware sine wave generator triggering into an analogue CRT monitoring loop.

## Tech Stack
- Sounddevice
- Numpy
- Pygame

## Setup
Ensure you have installed the global packages (`pip install -r requirements.txt`). 
```bash
source ../../.venv/bin/activate
cd days/day-07-keyboardoscilloscope
python keyboardoscilloscope.py
```
